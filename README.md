A real-time object detection and pose estimation tool that tracks people's skeletons from live video streams and sends the results over the network using [OSC](https://en.wikipedia.org/wiki/Open_Sound_Control) (Open Sound Control).

## Credits

Developed by the [Société des Arts Technologiques](https://www.sat.qc.ca) as a custom application built on [ossia score](https://ossia.io)
