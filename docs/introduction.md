LivePose offers a simple way to run computer vision models on live video input and send the results to any [OSC](https://en.wikipedia.org/wiki/Open_Sound_Control)-compatible application. Simply load a model, select your input feed, and start streaming data.

<video src="images/livepose-mdbooks-video.mov" autoplay loop muted></video>

## Quick start

1. Launch LivePose and select your camera
2. Load an `.onnx` model
3. Configure your OSC output and start streaming

For more details, see [Installation](installation.md) and [Usage](usage.md).
